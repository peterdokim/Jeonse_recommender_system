## 2. 전세 안전 점수 — 6개 차원

**점수 근거: 서울 전체 데이터 분포의 percentile 기반**
- 100점 = 상위 20% (P80 이상)
- 75점 = 상위 40% (P60~P80)
- 50점 = 중위 (P40~P60)
- 25점 = 하위 40% (P20~P40)

## 3. 탭별 상세 기능

### 탭 1: 전세 안전 진단 🏥

**사용자 입력:** 구 → 동 → (단지) 선택 + 보증금 금액

**출력:**
- 종합 등급 (A~D) + 6개 차원별 점수 카드
- 개인화 손실 위험: "보증금 5억 기준, 과거 하락 패턴상 최대 X만원 손실 위험"
- Cortex AI 해설 (자연어)

### 탭 2: 위기 시뮬레이션 ⚠️

**사용자 입력:** 탭1에서 선택한 동네/단지 + 보증금 (자동 연동)

**출력:**
- 2022년 하락 재현 시 이 동네 전세가 하락폭 (과거 실증)
- 보증금 기준 손실 금액
- 깡통전세 가능성 (전세가율이 100% 넘는지)
- **비슷한 전세가 범위(보증금 ±10%)에서 등급이 더 높은 동네 추천**

주의: "예측"이 아니라 "과거에 이 정도까지 떨어진 적 있음" 표현. 면책 문구 필수.

### 탭 3: 동네 비교 📊

**사용자 입력:** 2~3개 동네 선택

**출력 차트:**
- 전세가 추이 비교 (리치고 REGION 시세 시계열)
- 전세가율 변화 비교 (리치고 매매+전세)
- 6개 차원 레이더 차트 비교
- 업종별 소비 패턴 비교 — 필수소비 vs 선택소비 (SPH CARD_SALES 20개 업종)

### 탭 4: 동네 심층 분석 🔍

**사용자 입력:** 동네 1개 선택

**출력 섹션:**

| 섹션 | 데이터 | 내용 |
|------|--------|------|
| 시세 트렌드 | 리치고 REGION 시세 | 매매가+전세가 시계열 + 전세가율 변화 |
| 인구 트렌드 | 리치고 POPULATION_MOVEMENT + SPH 유동인구 | 전입/전출/순이동 + 거주/근무/방문 인구 |
| 연령별 이동 | 리치고 GENDER_AGE5 | 20~30대가 빠져나가고 있는가? |
| 소비 트렌드 | SPH CARD_SALES | 업종별 소비 히트맵, 필수소비 안정성 |
| 주민 프로필 | SPH ASSET_INCOME | 평균 자산/소득/대출/연체율/신용점수 |
| 교통 | 리치고 TRAIN_DISTANCE | 단지별 지하철 접근성 + 역 이름 + 승하차 |
| 이사 수요 | 아정당 V01/V05/V06 | 통신계약/인터넷신규설치/렌탈 트렌드 |

## 6. 주의사항

- 점수 기준은 **서울 전체 데이터 분포의 percentile** 기반 (임의 구간 아님)
- 가중치 근거: HUG 실제 사고율과 각 지표 상관분석 (전세가율이 가장 높은 상관)
- 과거 하락폭은 **"예측"이 아니라 "과거 실적"** 표현
- HUG는 **구 단위** (동 단위 아님)
- 아정당은 **시군구 단위** (법정동 없음)
- 전입전출은 **시군구 단위**
- 단지 데이터에 **평형 정보 없음** → 평당 가격 기준으로 진행
- 탭2 대안 추천: **사용자 보증금 ±10% 범위**에서 실제 전세가가 맞는 동네만 추천
- 면책 문구 필수: "본 서비스는 투자 조언이 아닌 참고 정보입니다"
- HUG 공공데이터 사용: 라이선스 링크/사본 포함 필수 (규칙 4.5절)
